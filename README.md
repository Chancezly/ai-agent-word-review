# AI Agent 术语复习

基于 Excel 文档《AI_Agent学习术语与8周复习计划》的网页闪卡复习工具。

## 功能

- **222 个术语**闪卡复习（英文 → 翻转看中文、解释、掌握标准）
- **自动间隔复习**：学后第 2 / 4 / 7 / 14 天安排待复习
- **8 周学习规划**：按当前周展示新词与周计划
- **日历打卡**：记录每日学习，高亮当前学习周
- **进度条**：总体掌握率 + 每周进度

## 启动

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址即可。

## 部署

项目已配置 GitHub Actions 自动部署到 GitHub Pages。

访问地址：https://chancezly.github.io/ai-agent-word-review/

- 术语数据来自 `src/data/content.json`（由 Excel 导出）
- 学习进度保存在浏览器 `localStorage`，无需登录

## 使用建议

1. 打开「复习」→ 选「今日全部」或「本周新词」
2. 先看英文，回忆后再翻转
3. 根据掌握程度点「不认识 / 认识 / 已掌握」
4. 在「概览」和「日历」查看进度
